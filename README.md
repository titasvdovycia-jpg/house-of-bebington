# House of Bebington - Automated Hedge Betting Scanner

A high-performance, browser-based scanner for sports arbitrage and margin betting. Built with a premium glassmorphic UI and real-time Telegram integration.

## 🚀 Getting Started

1. Visit the live link (once deployed).
2. Go to the **Settings** tab.
3. Enter your **The Odds API Key** and your **Telegram Token/Chat ID**.
4. Click **Save Settings**.
5. Return to the dashboard and click **Scan Live Markets**.

## 📊 Features

- **Live Arbitrage Detection**: Instant calculation of implied probability and profit margins.
- **Top 3 Telegram Reports**: Consolidated reports sent straight to your phone.
- **Global Coverage**: Supports 10+ sports, including NBA, Premier League, Tennis, and Cricket.
- **Token Health Meter**: Monitors your API usage to keep you within free limits.
- **Zero-Build Architecture**: Runs entirely in the browser using Vanilla JS.

## 🔒 Security

Your API keys and tokens are stored **exclusively** in your own browser's `localStorage`. They are never sent to any server other than the official The Odds API and Telegram Bot API.
